Setup a new Plugin
-------------------

0. Root to Static Folder

1. Install all NPM packages with
> npm install

(2. Just if you haven't installed the typings on your workstation yet)
> npm install typings -g

3. Install typings for Typescript
> typings install

(4. Just if you haven't installed webpack on your workstation yet)
> npm install webpack -g

5. Build your files with webpack
> webpack
